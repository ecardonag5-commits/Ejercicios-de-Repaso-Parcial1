#**Organización de los ejercicios**
En estos ejercicios de Repaso decidí implementar los 20 ejercicios dentro de un solo programa. (Con ayuda de la IA) hice un menú que permite seleccionar cada ejercicio mediante una opción numérica. Cada ejercicio fue colocado en su propio método para mantener el código ordenado y facilitar su lectura.

Así me permite ejecutar todos los ejercicios desde un mismo programa sin necesidad de crear múltiples archivos o programas independientes.
